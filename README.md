* Overview

The Employee Evaluation System is a web-based performance appraisal platform designed to streamline the employee review process within an organization. It provides an intuitive, step-by-step workflow for employees to complete their self-assessments, track their performance, and manage evaluation history across different years. The system improves transparency, reduces manual paperwork, and ensures that both employees and supervisors can efficiently track performance progression.

* Key Features
1. Yearly Evaluation Dashboard

Displays all available evaluations grouped by year.

Shows the status of each evaluation.

Quick access to detailed evaluation forms.

Clean UI with a left-panel navigation menu for easy switching between modules.

* User Interface Highlights

Modern, minimal UI with clear typography and structured content blocks.

Card-style sections for readability and ease of navigation.

Responsive layout, suitable for desktops and tablets.

Consistent iconography and color coding, especially for ratings (ex. green for Outstanding, red for Unsatisfactory).

Smooth page-to-page transitions for the multi-step form.

* Technical Implementation

Stack: Canvas apps

Database: MySQL, SharePoint

Authentication: Role-based access (Employee, Supervisor, Admin)

Additional Features:

Form validation

Secure session handling

Dynamic dropdowns

Evaluation record management
